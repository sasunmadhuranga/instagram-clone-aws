Instagram Clone (MERN Stack)

A full-stack, responsive social media platform inspired by Instagram, built using the MERN stack and Tailwind CSS. This application replicates core social networking features while delivering a modern, scalable, and user-friendly experience across devices.

Tech Stack

Frontend
•	React.js
•	Tailwind CSS
•	React Router

Backend
•	Node.js
•	Express.js
•	MongoDB
•	JWT Authentication

Deployment
•	Vercel (Frontend)
•	Render (Backend)

✨ Features

🔐 Authentication
•	Secure user registration and login
•	JWT-based authentication system
•	Persistent sessions and protected routes

🧭 Navigation
•	Smooth routing between:
    o	Home / Feed
    o	Profile
    o	Search
•	Responsive navigation UI

🔍 Search & Discovery
•	Search for users
•	View user profiles
•	Follow / unfollow functionality

🤝 Recommendations
•	Suggested users displayed in sidebar
•	Quick follow system for user discovery

👤 Profile Management
•	Edit profile details
•	Update profile picture
•	View followers & following lists

📝 Post Management
•	Create posts with:
    o	Captions
    o	Locations
•	Manage post visibility and interactions

❤️ Engagement
•	Like and unlike posts
•	Comment on posts
•	Save posts for later

📂 Saved & Shared Posts
•	View saved posts
•	Manage shared content from profile

📁 Project Structure
client/     → React frontend
server/     → Express backend
